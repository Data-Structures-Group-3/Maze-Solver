Detailed Description

Renders a Maze into a Swing window and exposes UI controls for stepping solver execution.

The renderer owns:

  • Grid drawing (cells + walls)
  • Control panel (Start, Pause, Restart)
  • Runtime controls (algorithm selector, step delay, traversed counter)
  • Automatic restart trigger when algorithm selection changes

◆ showMaze()

static void MazeRenderer.showMaze ( Maze maze)
static

Displays the maze in a GUI window.

Cell color mapping:

  • Start: green with S
  • Goal: blue with G
  • Blocked/unoccupied: dark gray
  • Traversable: white

Edge color mapping:

  • Black edge: wall or outer maze boundary
  • Light edge: passable adjacency
Parameters
mazemaze to render; expected to have dimensions initialized and path cells represented as non-null nodes
