Detailed Description

Incremental maze solver that supports DFS, BFS, and A*.

The solver keeps internal state between calls to updateSolve(), so callers can visualize or inspect each step one node at a time.

Typical usage: create a solver, set algorithm and maze, then call updateSolve() repeatedly until isFinished() returns true. Example:

MazeSolver solver = new MazeSolver();
solver.setMaze(maze);
solver.setAlg("A*");
while (!solver.isFinished()) {
solver.updateSolve();
}
if (solver.isSolved()) {
List<Point> path = solver.getFinalPath();
}
