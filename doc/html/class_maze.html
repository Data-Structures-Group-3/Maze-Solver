Detailed Description

Core maze model storing occupancy, neighbor links, and start/goal nodes.

The maze uses a rectangular grid of Node objects. Traversable cells contain a Node reference and blocked cells are null. Neighbor links represent valid movement between adjacent traversable cells.

Example import format for importFromFile(String) is: row count, column count, start: row,col, end: row,col, then one layout row per maze row using 1 (path) and 0 (wall). Example:

4
5
start: 3,0
end: 0,4
11111
10001
10101
11111

Typical implementation flow:

Maze maze = Maze.importFromFile("src/test_346x346.txt");
Maze.NeighborValidationResult vr = maze.validateNeighborConnections(25);
if (!vr.isValid()) {
for (String issue : vr.getIssues()) {
System.out.println(issue);
}
}
maze.exportToFile("out.txt");

static Maze Maze.generateRandomMaze ( int length,
int width )
static

Builds a random, guaranteed-solvable maze with the given dimensions.

The generated maze keeps the outer border blocked and carves interior corridors using a depth-first backtracking strategy to produce a classic maze appearance. For even dimensions, a follow-up pass extends corridors into the final interior row/column so right/bottom edges are not left as persistent solid strips.

Parameters
lengthtotal number of rows (minimum 3)
widthtotal number of columns (minimum 3)
Returns
newly generated solvable maze
Exceptions
IllegalArgumentExceptionif dimensions are smaller than 3

◆ importFromFile()

static Maze Maze.importFromFile ( String filePath) throws IOException
static

Import a maze from text using this format: line 1: row count line 2: column count line 3: start: row,col line 4: end: row,col line 5+: layout rows of 1s and 0s

Cells marked 1 are created through createCell(int, int, int) so neighbor links are built incrementally and remain bidirectionally consistent.

Parameters
filePathpath to the maze text file
Returns
newly created Maze instance populated from file data
Exceptions
IOExceptionif file reading fails
IllegalArgumentExceptionif file contents are invalid
